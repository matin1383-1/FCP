Course Syllabus

Foundation of Computer and Programming

Course Information

  • Course Title: Foundation of Computer and Programming
  • Institution: Shiraz University
  • Department: Mathematics and Computer Science
  • Credits: 3
  • Prerequisites: None
  • Professor name: Dr.Kazemi
  • Graders: Martin Auobii, Yoosof Rahimi, Pouria Safari, Shahryar Ahmadi, Kian Rahimi

Course Description

This course introduces students to the fundamental concepts of computing, algorithmic thinking, and the basics of programming. Students will learn how to approach problem-solving through code and develop the essential tools and mindset needed to think like a computer scientist.

Learning Objectives

  • Understand fundamental computer science concepts
  • Develop algorithmic thinking skills
  • Learn basic programming concepts and syntax
  • Practice problem-solving through programming
  • Understand data structures and algorithms basics
  • Develop debugging and testing skills

Course Schedule

Introduction to Computing
History of computing, computer hardware basics, software concepts
Programming Fundamentals
Variables, data types, operators, input/output
Control Structures
Conditional statements, loops, decision making
Functions and Modules
Function definition, parameters, return values, code organization
Data Structures
Arrays, lists, strings, basic data manipulation
Algorithms
Sorting, searching, algorithm efficiency
File Handling & Advanced Topics
File I/O, error handling, introduction to object-oriented programming
Review and Final Project
Course review, final project development and presentation

Assessment and Grading

Component Weight Description
Assignments 25% Homeworks and quizzes , Final project
Midterm Exam 35% Written exam covering first half of course
Final Exam 40% Comprehensive written and practical exam
